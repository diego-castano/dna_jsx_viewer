◆ Pentagram Creative Director 11 — Brand Rules

Do's & Don'ts

Protecting DNA's identity while applying Apple refinement.

✓ Do

Use generous whitespace. Let content breathe — Apple never cramps.

✕ Don't

Pack elements together. If you can't see the grid breathing, add space.

✓ Do

Use organic asymmetric corners on feature cards — DNA's signature.

✕ Don't

Use asymmetric corners on small UI elements like buttons or inputs.

✓ Do

Use Playfair Display for headlines — it's DNA's editorial voice.

✕ Don't

Use Playfair for body text or UI labels. DM Sans handles those.

✓ Do

Apply glassmorphism on nav and floating panels over gradients.

✕ Don't

Use glass on every component. It's for elevated floating contexts only.

✓ Do

Use toast notifications (pill) for quick feedback. Appear, inform, go.

✕ Don't

Use blocking modals for simple confirmations. Gabby hates "big blocks".

✓ Do

Use rgba() separators. Borders should be barely visible, never heavy.

✕ Don't

Use solid hex borders like #E6E4DD. They feel heavy vs rgba(3,0,0,.08).
